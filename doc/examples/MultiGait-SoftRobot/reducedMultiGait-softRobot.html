Multigait soft robot controlled by keyboard


How to use the demo

By default a PythonScriptController "waveController" moves the robot forward applying a specific sequence.

In the file reducedMultiGait-softRobot.pyscn, you may also comment line 85 and uncomment line 84 to use the "keyInteractiveController" to inflate cavities as you like.

After clicking on "Animate" you can click in the 3d view then inflate/deflate cavities by pressing CTRL + '+'/ CTLR + '-' .

To select the cavity to inflate, press CTRL + '0' (center cavity), CTRL + '1' (rear left cavity), CTRL + '2' (rear right cavity), CTRL + '3' (front left cavity), CTRL + '4' (front right cavity).
